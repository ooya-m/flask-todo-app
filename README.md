# タスク管理アプリ

日々のタスクを簡単に登録・管理できるシンプルなアプリケーションです。
<br>
<br>
<img width="1530" height="591" alt="アプリ画面大" src="https://github.com/user-attachments/assets/a6fc2b7a-1962-4a54-a19b-d76db4d8582a" />


## 開発環境
- **フロントエンド**：JavaScript / HTML / CSS
- **バックエンド**：Python / Flask
- **データベース**：MariaDB
  
<br>

## 主な機能
- **タスクの登録**：タスク、担当者、期限、備考を入力して保存できます。
- **タスクの一覧表示**：登録したタスクが下に並び、一目で確認できます。
- **タスクの削除**：完了したタスクを削除することができます。
